Checklist

A simple command-line checklist application written in C++.

This project was built as a learning exercise to practice core C++ programming concepts, with a focus on file handling and implementing basic CRUD (Create, Read, Update, Delete) operations.

Features

- Create checklist items
- View all items
- Update existing items
- Delete items
- Save data to a file
- Automatically load saved data on startup

Concepts Practiced

- File I/O
- CRUD operations
- Functions
- Loops and conditionals
- User input handling
- Basic program structure

Purpose

This project is part of my journey learning C++ by building small, practical applications instead of only solving exercises.

The goal was to understand how to persist data using files while creating a simple program that users can interact with from the command line.
