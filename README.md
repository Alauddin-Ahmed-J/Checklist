A simple command-line checklist application written in C++.
This project was built as a learning exercise to practice core C++ programming concepts, 
especially working with files and implementing basic CRUD (Create, Read, Update, Delete) operations.
Features
Create checklist items
View all items
Update existing items
Delete items
Save data to a file
Load saved data automatically
Concepts Practiced
File I/O
CRUD operations
Functions
Loops and conditionals
User input handling
Basic program structure
Purpose
This project is part of my journey learning C++
by building small, practical applications instead of only solving exercises.
